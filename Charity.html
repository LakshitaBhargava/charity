Ques 2.
(i) K nearest neighbor: Used in airplanes to warn pilots and air traffic if something is wrong on an airplane. It can be used in many system or biological health monitoring to notify technicians if anything is going wrong. The reason for selecting this algorithm for this project is because it is the simplest model and works on all the features given and also takes negligible time in training the system. Advantages- Simple to implement, Flexible to features, Can quickly respond to changes in input Disadvantages- More computational time on larger dataset, localized data effects the output more than the generalized data. (ii) Support Vector Machine: It is one of the best algorithms of supervised learning. The algorithm works well all kinds of data and best with noiseless data. It works wth a kernel trick to transform the data then finds an optimal boundary for the output. It is used here because can capture much more complex relationships between the data on its own. Advantages- High accuracy, works with non linear data also. Disadvantages- Time consumed is more when there are a lot of support vectors, Overfits when dataset is noisy. (iii) Random Forest: Used in places where many features are used to classify. It can be used in this as dataset is large and also it is better to have a fully connected relationship for this project. Advantages- Can work with many features, does not overfit, works well with large dataset. Disadvantages- Difficult to interpret, it is complex and consumes more time.

def train_predict(learner, sample_size, X_train, y_train, X_test, y_test): 
    '''
    inputs:
       - learner: the learning algorithm to be trained and predicted on
       - sample_size: the size of samples (number) to be drawn from training set
       - X_train: features training set
       - y_train: income training set
       - X_test: features testing set
       - y_test: income testing set
    '''
    
    results = {}
    
    # TODO: Fit the learner to the training data using slicing with 'sample_size' using .fit(training_features[:], training_labels[:])
    start = time() # Get start time
    learner.fit(X_train[:sample_size] , y_train[:sample_size])
    end = time() # Get end time
    
    # TODO: Calculate the training time
    results['train_time'] = end-start
        
    # TODO: Get the predictions on the test set(X_test),
    #       then get predictions on the first 300 training samples(X_train) using .predict()
    start = time() # Get start time
    predictions_test = learner.predict(X_test)
    predictions_train = learner.predict(X_train)
    end = time() # Get end time
    # TODO: Calculate the total prediction time
    results['pred_time'] = end-start
            
    # TODO: Compute accuracy on the first 300 training samples which is y_train[:300]
    results['acc_train'] = accuracy_score(y_train,predictions_train)
        
    # TODO: Compute accuracy on test set using accuracy_score()
    results['acc_test'] = accuracy_score(y_test,predictions_test)
    
    # TODO: Compute F-score on the the first 300 training samples using fbeta_score()
    results['f_train'] = fbeta_score(y_train,predictions_train,beta=beta)
        
    # TODO: Compute F-score on the test set which is y_test
    results['f_test'] = fbeta_score(y_test,predictions_test,beta=beta)
       
    # Success
    print "{} trained on {} samples.".format(learner.__class__.__name__, sample_size)
    
    predictions_train = learner.predict(X_train[:300])
    print (predictions_train)    
    # Return the results
    return results

Ques 3. Random forest works well on large dataset and outliers too. It also reduces variance which is why it can be used on this dataset as the data has high variance. Also when we look at the above "Accuracy score on testing graph" accuracy is better and almost contant in random forest which is what we need. And time taken for model predicting is minimum. The features in the dataset are independent of each other which is why random forest is used.

Ques 4. Random forest is machine learning algorithm which is flexible, easy to use and gives correct results most of the time. Random forest takes more time to give result as it has to create many different trees but almost everytime gives an accurate prediction. Random forest is like bootstrapping algorithm with Decision tree (CART) model. Random forest tries to build multiple CART model with different sample and then makes the final prediction by taking mean of each prediction.

For example, random forest can be used when classifying a fruit, a fruit can be classified by its color, taste, shape, size etc. Random forest will create different decision trees. Each tree is a little different than the other by features. Whichever tree has the most appropriate value is chosen. Similarly in this project the value of age, education_num etc are all different for different observation, therefore, making different trees is a better and easy way to predict. Also the features are independent of each other and random forest is best when there are independent features.
